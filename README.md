```
# Get the latest release of brain-games
$ npm install brain-games-ivan -g 
```

# Play
```
# Start game:
$ brain-games
$ brain-even
$ brain-calc
$ brain-gcd
$ brain-balance
$ brain-progression
$ brain-prime
```
