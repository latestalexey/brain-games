# Installation
```
# Get the latest stable release of brain-games
$ npm install brain-games-ivan -g 
```

# Play
```
# Start brain game
$ brain-games
```

```
# Start even game
$ brain-even
```

```
# Start calc game
$ brain-calc
```

```
# Start find gcd game
$ brain-gcd
```

```
# Start balance game
$ brain-balance
```

```
# Start progression game
$ brain-progression
```

```
# Start prime game
$ brain-prime
```
